# test-all

> Vue前端，后端，数据库练习

## 安装

### 安装前端依赖
```angular2html
npm install --save
```

### 进入service文件夹
```angular2html
npm install
```

### 启动项目
```angular2html
npm run dev
```

### 启动后台
```angular2html
node index.js
```

### 查看登录页面debug
> 在浏览器控制台里,输入下面命令，回车，刷新页面
```angular2html
localStorage.debug='login'

```
